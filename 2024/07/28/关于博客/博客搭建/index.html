环境准备

安装nodejs

  • 下载网址：Node.js 中文网 (nodejs.cn)
  • 下载安装程序后，一路next即可，然后安装到指定目录自己根据情况修改

安装git

  • git下载网址：Git (git-scm.com)

  • 然后选择如下进行下载

  • 然后一路next下去，安装完成后使用win + R键，然后cmd进入windows终端，输入node -v和git --version查看这两个应用是否安装成功并且是否自动配置了环境变量。如果没有设置环境变量，就搜索一下如何配置node和git的环境变量

  • 上述安装成功，那么环境就安装成功了

注册GitHub账号并且创建仓库

  • GitHub官网：GitHub: Let’s build from here · GitHub

  • 然后输入邮箱和密码，然后你的用户名

  • 之后GitHub会向绑定的邮箱发生一个邮件验证码，输入验证码即可

  • 到达用户界面后点击创建一个新的仓库

  • 然后为仓库取名，仓库的命名格式为GitHub用户名.github.io，如果不这样命名可能会出现网页打不开的问题。

  • 之后点击继续按钮
  • 然后使用SSH连接，并将仓库地址复制下来。后续会方便一点

post文档

  • post文档就是创建一个新的博客文章，博客文章的默认配置就在这边，使用hexo new "文章标题"即可创建一个新的博客文档

  • 自己可以添加东西

  • 下次新创建一个md文档就会出现如下模式

  • 下面逐个介绍
名称 作用和功能
title 文章标题
katex 数学公式修饰插件
mathjax 数学公式修饰插件
data 文章创建日期
tags 标签
description 前言
cover 顶部图
top_img 文章的封面图（可能要在特定主题才会显示文章封面图）
categories 分类
sticky 置顶或者放置在第二篇文章等

page文档

  • page文档是用于创建独立的页面，用于跳转
  • 比如我创建了一个读书清单的页面，使用hexo new page 'page-name'即可创建一个独立的页面

  • scaffolds目录下的page文档是创建页面时的默认配置，打开后会出现这个配置，创建新的页面时，默认配置就会这样
